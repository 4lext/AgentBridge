# How to Contribute

We're thrilled you're interested in helping with AgentBridge! This is a new open-source project aimed at making Chrome Native Messaging accessible to all developers.
## How You Can Help

The best way to help right now is to:

-    Open an Issue: If you find a bug, have a feature request, or want to suggest an improvement, please use our issue templates.

-    Provide Feedback: Test the app on your platform (especially Windows and Linux) and report any bugs you find.

-    Improve Documentation: If you find the README.md or other docs confusing, feel free to open a PR with improvements.

Getting Started: Building From Source

The project is built with:

  * Electron

  * TypeScript

  * React

  * Vite

To get started with development:

    Clone the Repository
```bash
    git clone [https://github.com/your-org/agent-bridge.git](https://github.com/your-org/agent-bridge.git)
    cd agent-bridge
```
    Install Dependencies
```bash
    npm install
```
    Run the App in Development Mode
```bash
    npm run electron:dev
```
    This will launch the Vite dev server for the React UI and the Electron app simultaneously, with hot-reloading enabled.

Submitting a Pull Request

  1  Fork the repository.

  2  Create a new branch for your feature (git checkout -b feature/my-new-feature).

  3  Make your changes.

  4  Ensure your code lints and builds (npm run build).

  5  Open a Pull Request with a clear description of what you've done.

We adhere to a standard code of conduct. Please be respectful and constructive in all your interactions.
